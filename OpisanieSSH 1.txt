1) Ставим все настройки контейнера на стандартные
2-3) Создаём 2 контейнера lxc
4-5) Запускаем их
6) Проверяем их состояние и смотрим ip адресс
7-8) Удаялем содержимое директории .ssh, чтобы не было никаких конфликтов 
9) Создаём rsa ключ
Чтобы не допустить никаких ошибок бдуем передавать ключ через ssh-copy-id, для этого производим действия 10-21
10) Заходим на контейнер dz1 
11-12) Переходим в файл ssh_config, изменяем там строчки 'PermitRootLogin prohibit-password' на 'PermitRootLogin yes'. 'PasswordAuthentication no' на 'PasswordAuthentication yes'
13) Перезапускаем сервис
14) Меняем пароль root, так как пароль изначальный мы не знаем, чтобы можно было передать ключ через ssh-copy-id
15) Выходим из контейнера
16-21) Повторяем всё тоже самое для dz2
22-23) Передаём ключ на dz1 и dz2
24) Подключаемся через ssh без ввода пароля к dz2 
25) Всё успешно, выходим
26) Подключаемся через ssh без ввода пароля к dz1
27) Создаём ключ rsa
28) Передаём его на dz2
29) Подклбчаемся через ssh без ввода пароля к dz2, находясь при этом в dz1, всё успешно
